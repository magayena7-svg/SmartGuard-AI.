## Tampilan Dashboard SmartGuard AI
![Dashboard Preview](Cuplikan%20layar%202026-04-24%20030735.png)

### Deskripsi Proyek
Proyek ini dikembangkan untuk mendeteksi transaksi mencurigakan secara real-time menggunakan algoritma Machine Learning. Dashboard ini menampilkan:
* **Simulasi Input Transaksi:** Memungkinkan pengguna memasukkan jumlah uang dan lokasi.
* **Hasil Deteksi AI:** Memberikan status keamanan transaksi (Aman/Mencurigakan).
* **Monitoring Grafik:** Visualisasi volume transaksi menggunakan Chart.js.
